使用真实 API 的解决方案 ：

由于微信小程序环境的限制（缺少必要的加密 API 和域名限制），直接在小程序中调用火山引擎 API 会遇到问题。最可靠的解决方案是使用服务器端代理：

1. 在服务器端创建代理接口 ：
   
   - 创建一个 API 端点，例如 https://your-server.com/api/ai-optimize
   - 这个端点需要接收 imageBase64 和 prompt 参数
   - 在服务器端使用正确的加密算法生成签名
   - 调用火山引擎的 API
   - 将结果返回给微信小程序
2. 配置服务器端代理地址 ：
   
   - 修改 src/api/ai-optimize.ts 文件中的 SERVER_PROXY_URL 为你的服务器端代理地址
3. 服务器端实现 ：
   
   - 使用 Node.js 或其他后端语言实现
   - 使用 crypto 模块来生成 SHA256 和 HMAC-SHA256
   - 调用火山引擎 API 并返回结果
这样，微信小程序就不需要在客户端处理加密和 API 调用，避免了环境限制的问题。

注意 ：你需要确保服务器端代理地址已经添加到微信小程序的合法域名列表中。
